## 常用正则表达式

匹配手机号码
```
^(13[0-9]|14[5|7]|15[0|1|2|3|5|6|7|8|9]|18[0|1|2|3|5|6|7|8|9])\d{8}$
```

匹配电话号码
```
\d{3}-\d{8}|\d{4}-\d{7}

```

匹配邮箱
```
^\w+([-+.]\w+)*@\w+([-.]\w+)*\.\w+([-.]\w+)*$
```

匹配身份证号码
```
15位
^[1-9]\\d{7}((0\\d)|(1[0-2]))(([0|1|2]\\d)|3[0-1])\\d{3}$

18位
^[1-9]\\d{5}[1-9]\\d{3}((0\\d)|(1[0-2]))(([0|1|2]\\d)|3[0-1])\\d{3}([0-9]|X)$
```

匹配URL
```
[a-zA-z]+://[^\s]*

或

^[http|https]://([\w-]+\.)+[\w-]+(/[\w-./?%&=]*)?$
```

匹配空行
```
\n[\s| ]*\r
```

匹配首尾空白
```
^\s*|\s*$
```

匹配邮政编码
```
[1-9]\d{5}(?!\d)
```

匹配QQ号码
```
[1-9][0-9]{4,}
```

必须包含大小写字母和数字的组合，不能使用特殊字符，长度在8-10之间
```
^(?=.*\d)(?=.*[a-z])(?=.*[A-Z]).{8,10}$
```

必须以字母开头，长度在6~18之间，只能包含字母、数字和下划线
```
^[a-zA-Z]\w{5,17}$
```

匹配是否含有^%&',;=?$\等字符
```
[^%&',;=?$\x22]+
```

匹配特定数字
```
^[0-9]*$                                      // 匹配数字
^\d{n}$                                       // 匹配n位的数字
^\d{n,}$                                      // 匹配至少n位的数字
^\d{m,n}$                                     // 匹配m-n的数字
^[1-9]\d*$                                    // 匹配正整数
^-[1-9]\d*$                                   // 匹配负整数
^-?[1-9]\d*$                                  // 匹配整数
^[0-9]+\.{0,1}[0-9]{0,2}$                     // 匹配整数或小数
^[1-9]\d*|0$                                  // 匹配非负整数（正整数 + 0）
^-[1-9]\d*|0$                                 // 匹配非正整数（负整数 + 0）
^[1-9]\d*\.\d*|0\.\d*[1-9]\d*$                // 匹配正浮点数
^-([1-9]\d*\.\d*|0\.\d*[1-9]\d*)$             // 匹配负浮点数
^-?([1-9]\d*\.\d*|0\.\d*[1-9]\d*|0?\.0+|0)$   // 匹配浮点数
^[1-9]\d*\.\d*|0\.\d*[1-9]\d*|0?\.0+|0$       // 匹配非负浮点数（正浮点数 + 0）
^(-([1-9]\d*\.\d*|0\.\d*[1-9]\d*))|0?\.0+|0$  // 匹配非正浮点数（负浮点数 + 0）
```

匹配特定字符串
```
^.{3}$                                        // 匹配长度为3的字符
^[A-Za-z]+$                                   // 匹配26个英文字母
^[A-Z]+$                                      // 匹配26个大写英文字母
^[a-z]+$                                      // 匹配26个小写英文字母
^[A-Za-z0-9]+$                                // 匹配数字和26个英文字母
^\w+$                                         // 匹配数字、26个英文字母或者下划线
^[\u4e00-\u9fa5]+$                            // 匹配中文
^[\u4E00-\u9FA5A-Za-z0-9_]+$                  // 匹配中文、英文、数字和下划线
^[\uFF00-\uFFFF]+$                            // 匹配全角符号
^[\u0000-\u00FF]+$                            // 匹配半角符号
```
